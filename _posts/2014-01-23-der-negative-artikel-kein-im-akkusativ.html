"kein" im Nominativ (否定词 "kein" 的第一格)

m Das ist ein Computer. -> Das ist kein Computer.
这是一台电脑。 这不是一台电脑。
n Das ist ein Telefon. -> Das ist kein Telefon.
这是一台电话。 这不是一台电话。
f Das ist eine Wohnung. -> Das ist keine Wohnung.
这是一套住房。 这不是一套住房。
Pl. Das sind Bücher. -> Das sind keine Bücher.
这些是书。 这些不是书。

"kein" im Akkusativ (可数名词)

m Ich habe einen Computer. -> Ich habe keinen Computer.
我有一台电脑。 我没有一台电脑。
n Ich habe ein Telefon. -> Ich habe kein Telefon.
我有一台电话。 我没有一台电话。
f Ich habe eine Wohnung. -> Ich habe keine Wohnung.
我有一套住房。 我没有一套住房。
Pl. Ich habe Bücher. -> Ich habe keine Bücher.
我有书。 我没有书。
  1. 在不定冠词前加上字母 k, 即可转换为否定句：einen -> keinen; ein -> kein; eine -> keine
  2. 零冠词的复数名词：keine

"kein" im Akkusativ (不可数名词)

m Ich habe Unterricht. -> Ich habe keinen Unterricht.
我有课。 我没有课
n Ich habe Geld. -> Ich habe kein Geld.
我有钱。 我没有钱。
f Ich habe Zeit. -> Ich habe keine Zeit.
我有时间。 我没有时间。
  • der Unterricht (阳性不可数名词)：否定词用 keinen
  • das Geld (中性不可数名词)：否定词用 kein
  • die Zeit (阴性不可数名词)：否定词用 keine

不可数名词否定形式的一般规律：

  • 不可数名词用 kein 做否定
  • 否定词 kein 根据不可数名词不同的词性作相应的变化

aber (例外):

Ich habe Geld. -> Ich habe kein Geld.
我没有钱。kein 否定零冠词名词
Ich habe viel Geld. -> Ich habe nicht viel Geld.
我有很多钱。 我没有很多钱。否定的是形容词 viel, 所以用 nicht。nicht 否定形容词。
