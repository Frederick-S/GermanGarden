单词讲解

1. die Beschreibung, -en 描述，描绘 Die beiden Bilder haben je eine Beschreibung. 这两幅图每一幅都有相应的描述。

beschreiben 说明，描述

Beschreiben Sie das Bild bitte. 请你描述一下这幅图片。

2. zeigen v. 画的是，展示 Das Bild zeigt einen Mann mit Hut. 图上显示了一个戴着帽子的男人。

Zeig mir deinen Pass bitte. 请出示一下你的护照。

3. spielen v. (此处为) 发生在... Die Szene spielt im Winter. 这个场景发生在冬天的时候。

Die Geschichte spielt im Sommer. 这个故事发生在夏天的时候。

4. diszipliniert Adj. 守纪律的，有纪律的 Mit der Uniform sieht er besonders ernst und diszipliniert aus. 他穿着制服，看起来特别严肃和守纪。

undiszipliniert 不守纪律的

ernst Adj. 严肃的

重点讲解

1. Es zeigt eine Szene aus dem Studentenleben um 1805.

1. jm. etwas zeigen: 给某人展示某事

Er hat mir den Brief gezeigt. / Sie zeigt großes Interesse für Geschichte.

2. um Präp: (时间) 大概在...的时候；(地点) 在...周围

Um das Dorf liegen die Felder.

2. Die Studenten tragen Uniform und reiten auf Pferden. Hier begleiten sie wohl eine Dame. Sie trägt einen schwarzen Mantel und sitzt in einem Pferdeschlitten.

1. tragen: 穿着，戴着 VS an/ziehen: 穿上

Zieh den Mantel an! Es ist kalt.

3. Die Szene spielt also im Winter. Die Gesellschaft auf diesem Bild wirkt heiter und fröhlich.

1. wirken v. 起作用

Das Medikament wirkt schnell.

4. Hier ist ein Student in der Uniform einer Studentenverbindung oder Burschenschaft zu sehen.

einer Studentenverbindung = von einer Studenenverbindung, …

Bildbeschreibung

1. 在图上有什么，在图上能看见什么

Auf dem Bild ist … abgebildet. (ein Baum)

Hier ist … zu sehen. (ein Baum)

Das Bild zeigt … (einen Baum)

Hier sieht man … (einen Baum)

2. 看起来，显得

wirken z.B. Die Personen machen eine Grillparty und sie wirken heiter und fröhlich.

aus/sehen z.B. Der Mann sieht aber ärgerlich aus.

3. 表示方位在哪一部分可以看到

Im Vordergrund / Im Hintergrund … 在前面部分，在后面部分

In der Mitte 在中间

oben 在上面

unten 在下面

Auf der linken Seite / Auf der rechten Seite … 在左边，在右边

以上方位词或短语可搭配: … sieht man / kann man … erkennen
